== Sencillo blog en RubyonRails

Para instalarlo es necesario:

- Tener rubyonrails instalado. Se ha realizado con ruby v1.9.3 y rails v3.2.8 

- Ejecutar un 'bundle install' para instalar las gem necesarias. Si diera problemas con mysql ejecutar
  lo siguiente: sudo apt-get install libmysql-ruby libmysqlclient-dev
  y volver a ejecutar 'bundle install' 

- crear en mysql un user y pass como los especificados en config/database.yml
